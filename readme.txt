!!! Экспортированные из базы данных файлы находятся в папке
12task.

Для корректного запуска необходимо:
1) Установка текущей базы данных. Установить базу данных c названием maxaderiha в качестве текущей, чтобы затем ее использовать. Для этого надо использовать команду use, после которой идет название базы данных. При этом не важно, существует ли такая бд или нет. Если ее нет, то MongoDB автоматически создаст ее при добавлении в нее данных.
2) Ипорт данных в текущую базу. Для импорта данных из файла articles_export.json использовать команду: mongoimport --db maxaderiha --collection articles --file articles_export.json, а для файла users-export.json, соответсвенно, - mongoimport --db maxaderiha --collection users --file users_export.json
3) Установка необходимых пакетов. Из корневой директории проекта
выполнить npm install (все депенденсис указаны в package.json). 4) Запуск сервера. Для запуска сервера выполнить команду node app.js из той же директории.
